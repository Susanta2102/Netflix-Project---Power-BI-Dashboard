# Netflix Power BI Dashboard Project

## Overview
This project involves creating an interactive Power BI dashboard using the Netflix dataset from Kaggle. The dashboard provides comprehensive insights into Netflix TV shows and movies, leveraging various Power BI features and visualizations.

## Features
- **Data Preparation and Ingestion**: Cleaning and transforming the dataset for analysis.
- **Data Modeling**: Structuring the data for efficient querying and visualization.
- **Funnel Chart**: Visualizing data flow in a funnel format.
- **Card with Reference Labels**: Displaying key metrics with reference labels.
- **Table with Images**: Incorporating images in tables for enhanced visualization.
- **Slicer Filter**: Adding interactive filters for dynamic data exploration.
- **Bar Chart with Stepped Line Axis**: Combining bar charts with stepped line axes for dual-axis visualizations.
- **Table with Databars and Conditional Formatting**: Enhancing tables with data bars and conditional formatting.
- **Q&A Section**: Allowing users to ask questions and get answers based on the data.

## Dataset
The dataset used in this project is from Kaggle and can be found [here](https://www.kaggle.com/datasets/snehaanbhawal/netflix-tv-shows-and-movie-list).
